# llmHub

Platform for creating and managing LLM agents with custom personas. Built as an engineering thesis project.

Each agent gets its own system prompt, personality, and conversation history — switch between agents and chat with different "personas" in one interface.

## Features

- Create and manage multiple AI agents with custom system prompts
- Chat with each agent independently, maintaining separate conversation histories
- Switch between personas in real time
- Dockerized setup — one command to run

## Tech stack

**Backend** — Python, FastAPI, LangChain  
**Frontend** — Angular, TypeScript  
**Infrastructure** — Docker Compose
